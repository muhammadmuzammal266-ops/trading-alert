** Game Tips and Tricks Platform**
**Overview**

The Game Tips and Tricks Platform is a web-based application that allows users to share and discover gameplay strategies in a structured and searchable format.

The system helps gamers quickly find useful tips without searching across multiple scattered sources.


**Technologies Used**

Node.js (Express)

MySQL 

Docker & Docker Compose

GitHub for version control

How to Run (Docker)

Ensure Docker Desktop is running


**Open in browser:**

App: http://localhost:3000

phpMyAdmin: http://localhost:8081

**Features**

View all game tips

Add new tips

Search tips by game name

Structured database storage
